# HashiCorp icons for draw.io

A shape library for [draw.io](https://www.diagrams.net/) with HashiCorp product icons. Self-contained — SVGs embedded in the `.xml` file, no external fetches.

## Contents

26 shapes: HashiCorp bug + compact strap; Terraform, Vagrant, Packer, Waypoint, Nomad, Vault, Boundary, Consul — each as LogoMark (icon only), full logo, and HCP variant where it exists; plus HCP Vault Secrets and HCP Vault Radar.

## Install

`File` → `Open Library...` → pick `HashiCorp.xml`. Or `File` → `Open Library from` → `URL` with a raw GitHub link.
